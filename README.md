# PostureGuard

PostureGuard is a lightweight Windows application that helps you build better posture habits using your webcam. Instead of reminding you on a timer, it monitors your posture in real time and only sends a notification when it detects you've been slouching for a while.

## Features

* Real-time posture monitoring
* Webcam face detection with OpenCV
* Personalized posture calibration
* Live posture score
* Windows notifications
* Automatically launches with Windows

## Built With

* C#
* .NET
* OpenCV
* Windows Forms

## Why I Made It

I noticed I would start slouching almost immediately whenever I sat down, especially while programming. Most posture reminder apps only use timers, so I decided to build one that actually watches my posture and reminds me only when I need it.

## Disclaimer

PostureGuard is meant to help build better posture habits. It is not a medical device or a replacement for professional medical advice.
